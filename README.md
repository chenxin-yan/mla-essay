# MLA Essay Template

> Markdown essay => MLA docx

## Dependencies

- [pandoc](https://pandoc.org)

## Usage

1. update `src/header.md` with relevant information in "`< >`"
2. add citations to `src/cite.bib`
3. write your essay in `essay.md`
4. run `make`
5. output can be found at `dist/essay.docx`
